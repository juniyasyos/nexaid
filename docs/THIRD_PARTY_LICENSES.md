# Third-Party Licenses

This project contains third-party dependencies and assets that are licensed separately from the project license.

The authoritative source for dependency licenses is the dependency manifests and lockfiles in the repository, including:

- [composer.json](../composer.json)
- [composer.lock](../composer.lock)
- [package.json](../package.json)
- [package-lock.json](../package-lock.json)

Maintain this document with a summary of any redistributed third-party components, their license names, and any required notices.

## Inventory

| Component | License | Notes |
|---|---|---|
| Laravel framework and ecosystem packages | See dependency manifests | Review upstream package licenses before redistribution |
| Frontend npm packages | See dependency manifests | Review upstream package licenses before redistribution |

## Notice Handling

When a third-party package requires attribution, include the required notice text here or reference the upstream license file.

If a bundled asset ships with its own license, store the corresponding notice alongside this document or in a subdirectory under `docs/`.
